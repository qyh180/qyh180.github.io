Selenium Server使用配置（一）

发表于 2020-07-22 更新于 2020-07-24 分类于 Python Valine：

1. Selenium Grid是什么？

Selenium Grid专门用于在不同的浏览器、操作系统和机器上并行运行多个测试用例。通过将命令发送给远程的Web浏览器实例来完成并行测试，其中的一台服务器充当中心。Hub会在多态计算机上并发执行测试，从而集中管理不同的浏览器，而不用对每个浏览器进行单独的测试，Selenium Grid使跨浏览器测试变得容易，因为单个测试能够在多台机器和浏览器上运行，快速完成测试。

Selenium Grid 的两个主要组件时：

  • Hub是一台中心服务器，它接受来自WebDriver客户端的访问请求，并将JSON测试命令路由到节点上的远程驱动器。它从客户端获取指令，并在各个节点上并发的远程执行它们。

  • Node是一个由本地操作系统和远程WebDriver组成的远程设备。它以JSON测试命令的形式接收来自中心的请求，并使用WebDriver执行它们

由于学生党比较穷，后面流程都是在一台本地计算机上进行的。

2. 整个流程

要使用Selenium Grid进行分布式的操作，步骤大致是下面几个。

  1. 在本地CMD创建hub
  2. 在hub创建node,要对node配置端口号
  3. 在python脚本中配置要执行哪一个Node

3. webdriver下载

webdriver 是用来执行并操作浏览器的一个API，程序脚本通过webdriver来直接对浏览器进行操作，相当于起到一个链接浏览器和程序的桥梁。

3.1 IE

  1. 下载地址：IEDriverServer
    选择和电脑位数相同的额压缩包
  2. 将压缩包内.exe文件放到Python安装目录下的Scripts\文件夹下即可。

    3.2 Chrome

  3. 下载地址：ChromeDriver
    注意要和电脑中Chrome版本一致，
  4. 将压缩包内.exe文件放到Python安装目录下的Scripts\文件夹下即可。

    3.3 Firefox

  5. 下载地址：geckodriver
  6. 将压缩包内.exe文件放到Python安装目录下的Scripts\文件夹下即可。

4. Selenium Grid启动

4.1 启动hub

java -jar selenium-server-standalone-3.149.0.jar -port 4455 -role hub

-port 4455是开启4455号端口-role hub是将Selenium server作为hub或者Grid server

  1. 重新启动一个hub 端口是4444
